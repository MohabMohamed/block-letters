# block-letters
Cool way of printing block letters of a string

Given an input string, it will output it with cool block letters!

Example:
```
$ ./block-letters "dab"
 ______   ______  _______
|  ___ \ |  __  ||  __   \
| |   \ \| |__| || |__| _/
| |   | ||  __  ||  __ |_
| |___/ /| |  | || |__|  \
|______/ |_|  |_||_______/
$ ./block-letters "beach"
 _______  ______  ______  _______  _     _
|  __   \|  ____||  __  ||  _____|| |   | |
| |__| _/| |____ | |__| || |      | |___| |
|  __ |_ |  ____||  __  || |      |  ___  |
| |__|  \| |____ | |  | || |_____ | |   | |
|_______/|______||_|  |_||_______||_|   |_|
```
