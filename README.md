# block-letters
Cool way of printing block letters of a string

Given an input string, it will output it with cool block letters!

Example:
```
$ ./block-letters "aba"
 ______  ________  ______
|  __  ||   __   \|  __  |
| |__| ||  |__| _/| |__| |
|  __  ||   __ |_ |  __  |
| |  | ||  |__|  \| |  | |
|_|  |_||________/|_|  |_|
$ ./block-letters "abaa"
 ______  ________  ______  ______
|  __  ||   __   \|  __  ||  __  |
| |__| ||  |__| _/| |__| || |__| |
|  __  ||   __ |_ |  __  ||  __  |
| |  | ||  |__|  \| |  | || |  | |
|_|  |_||________/|_|  |_||_|  |_|
```
